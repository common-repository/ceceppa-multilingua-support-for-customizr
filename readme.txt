Plugin to make Ceceppa Multilingua work with Customizr.

== Description ==

This plugin allow user to translate "Customizr slider" with "Ceceppa Multilingua"-

= Features =

- Translate Customizr fields: Title text, Description text, Button text 
- Translate featured post

= Requires =

Ceceppa Multilingua >= 1.4.10

= About =
For more Information visit the [Plugin Homepage](http://www.alessandrosenese.eu/en/ceceppa-multilingua/ceceppa-multilingua-for-customizr)

== Installation ==

Installation of this plugin is fairly easy:

1. Download the plugin from wordpress. 
1. Extract all the files. 
1. Upload everything (keeping the directory structure) to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Manage desidered languages
